# g0v 社群守則 (Code of Conduct, CoC)

這個 repo 是用於 g0v 的社群參與準則，可供 g0v 所有線上與線下活動使用。

目前版本：2023/4/5，參考[共編筆記](https://g0v.hackmd.io/COC)。
